# status — 记忆系统现状检查

**触发词**：检查记忆系统 / 健康检查 / 现状如何 / 有多少记忆

**前置条件**：无

---

## 检查项目

### 1. L1 事件层
- [ ] 事件总数
- [ ] 最近一次记录时间
- [ ] 待升华事件数量

### 2. L2 经验层
- [ ] 有效行动条目数
- [ ] 失败教训条目数
- [ ] 关键洞察条目数

### 3. L3 模式层
- [ ] 已识别模式数量
- [ ] 高置信度模式数量
- [ ] 中低置信度模式数量（需要补充证据）

### 4. L4 原则层
- [ ] 原则总数
- [ ] 有来源模式支撑的原则数量
- [ ] 有反例标注的原则数量

### 5. L5 身份层
- [ ] 核心身份陈述
- [ ] 价值主张
- [ ] 支撑原则数量

---

## 输出格式

```markdown
# 记忆系统现状

## 层次覆盖率

| 层次 | 内容数 | 状态 |
|------|--------|------|
| L1 事件层 | N | ✅/⚠️ |
| L2 经验层 | N | ✅/⚠️ |
| L3 模式层 | N | ✅/⚠️ |
| L4 原则层 | N | ✅/⚠️ |
| L5 身份层 | N | ✅/⚠️ |

## 待办项

- [ ] [具体待办]
- [ ] [具体待办]

## 建议

[根据检查结果给出改进建议]
```

---

## 判断标准

| 层次 | 状态 | 说明 |
|------|------|------|
| ✅ 健康 | 内容数 ≥ 3，且有定期更新 | 系统运转正常 |
| ⚠️ 薄弱 | 内容数 < 3 或 90 天未更新 | 需要补充 |
| 🔴 危险 | 内容数 = 0 | 需要立即补充 |

# Memory Layers — 五层记忆法

> 把每一次经历，变成持续增值的智慧资产。

## 这个 skill 做什么

帮你把零散的经历，通过五个层次逐步升华，形成一个自我强化的智慧系统：

| 层次 | 名称 | 问题 | 输出 |
|------|------|------|------|
| L1 | 事件层 | 发生了什么？ | 原始记录 |
| L2 | 经验层 | 学到了什么？ | 教训与洞察 |
| L3 | 模式层 | 有什么规律？ | 跨事件模式 |
| L4 | 原则层 | 指导原则是什么？ | 行动准则 |
| L5 | 身份层 | 我是谁？ | 核心身份定位 |

**核心理念**：知识必须被编译一次，然后持续维护。不是每次重新推导，而是逐层升华。

---

## 执行约束（红线）

1. **禁止跳过层次**：L2 经验不能凭空产生，必须有具体事件支撑
2. **禁止放之四海而准**：每条原则必须有来源事件或模式支撑
3. **禁止一次性完成**：五层是持续迭代的系统，不是一次性写作
4. **禁止脱离具体上下文**：原则必须绑定到具体领域/场景
5. **禁止 L4 原则复述 L3 模式**：原则必须比模式更具体、更有可操作性；如果只是换了词而没有深化，返工
